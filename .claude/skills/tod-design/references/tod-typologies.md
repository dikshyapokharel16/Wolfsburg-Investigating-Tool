# TOD Typologies -- Complete Design Specifications

This reference provides 8 distinct TOD typologies, each representing a
different combination of transit mode, urban context, and development
strategy. Use these typologies as starting templates and calibrate to the
specific site conditions.

---

## Typology 1: Urban Core TOD

**Definition**: Major interchange station in or adjacent to the central
business district. Highest intensity development. Multiple transit lines
converge, creating a regional destination.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | Metro interchange (2+ lines), often with commuter rail |
| Walk Catchment | 800m (often extended to 1000m+ due to pedestrian quality) |
| Minimum Density | 150 DU/ha (net) |
| FAR Range | 5.0 - 15.0 |
| Height Range | 12 - 60+ stories |
| Block Size | 60-100m x 60-100m |
| Street Width (ROW) | Arterials 30-40m, locals 15-20m |

### Density Targets

| Zone | FAR | Typical Use |
|---|---|---|
| Station core (0-200m) | 8.0 - 15.0 | Office towers, hotels, landmark retail, transit interchange |
| Inner ring (200-400m) | 5.0 - 8.0 | Mixed-use towers, residential + office + retail |
| Middle ring (400-600m) | 3.0 - 5.0 | Residential towers and mid-rise mixed-use |
| Outer ring (600-800m) | 2.0 - 3.5 | Mid-rise residential, community facilities |

### Use Mix

- Office: 35-45%
- Residential: 25-35%
- Retail / F&B: 10-15%
- Hotel: 5-10%
- Civic / Cultural: 5-10%
- Ground floor: 100% active frontage required on all streets

### Street Types

- **Primary commercial boulevard**: 36-40m ROW, transit median, protected bike lanes, wide sidewalks with arcades
- **Secondary mixed-use street**: 24-28m ROW, 2 travel lanes, bike lanes, generous sidewalks
- **Pedestrian priority street**: 12-20m ROW, shared surface, service access only
- **Station plaza**: minimum 0.5 ha, car-free, direct connection to all station entrances

### Parking Strategy

- Zero minimum parking requirement
- Maximum 0.2 spaces per residential unit, 1.0 per 100 m2 office
- 100% underground, no visible parking structures
- Comprehensive TDM: car-share, bike-share, transit pass, delivery consolidation

### Global Precedents

- **Tokyo Marunouchi**: 120 ha around Tokyo Station, FAR up to 13.0, seamless rail-building integration
- **Hong Kong Central/Admiralty**: MTR stations integrated with commercial towers, underground pedestrian network
- **Singapore Tanjong Pagar / Marina Bay**: MRT interchange, 15+ FAR, underground connections, mixed-use towers
- **London King's Cross**: 27 ha regeneration around interchange, 50 buildings, 2,000 homes, mixed-use
- **Paris La Defense**: RER/Metro interchange, commercial district with 3.6M m2 of offices

## Typology 2: Urban Neighborhood TOD

**Definition**: Standard metro or tram station in an established urban
neighborhood. Medium-high intensity development that reinforces existing
neighborhood character while increasing density.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | Metro (single line) or LRT/Tram |
| Walk Catchment | 600-800m |
| Minimum Density | 80 DU/ha (net) |
| FAR Range | 2.0 - 5.0 |
| Height Range | 4 - 12 stories |
| Block Size | 70-120m x 70-120m |
| Street Width (ROW) | Collectors 22-28m, locals 15-20m |

### Density Targets

| Zone | FAR | Typical Use |
|---|---|---|
| Station core (0-200m) | 4.0 - 5.0 | Mixed-use, neighborhood center, small-format retail |
| Inner ring (200-400m) | 2.5 - 4.0 | Mid-rise residential with ground-floor retail on main streets |
| Middle ring (400-600m) | 1.5 - 2.5 | Mid-rise residential, townhouses on quieter streets |
| Outer ring (600-800m) | 1.0 - 1.5 | Low-rise residential, row houses, community facilities |

### Use Mix

- Residential: 55-65%
- Retail / F&B: 10-15%
- Office / Co-working: 5-10%
- Civic / Community: 10-15% (school, library, health center, community center)
- Ground floor: 80% active on main streets, residential entries on side streets

### Street Types

- **Main street / high street**: 24-28m ROW, 1-2 lanes per direction, on-street parking, protected bike lanes, wide sidewalks with street trees
- **Residential collector**: 18-22m ROW, 1 lane per direction, bike lanes, tree-lined sidewalks
- **Local residential street**: 14-18m ROW, shared or low-speed, traffic calmed
- **Neighborhood square**: 0.1-0.25 ha, adjacent to station, daily market or playground

### Parking Strategy

- Maximum 0.5 spaces per unit within 400m, 0.8 outside
- Structured parking wrapped with residential or retail
- Shared parking between residential and commercial
- On-street parking managed with meters and time limits

### Global Precedents

- **Paris Metro neighborhoods** (Oberkampf, Bastille): fine-grain, mixed-use, 6-8 stories, lively streets
- **Barcelona Eixample**: tram/metro stops in superblock grid, 5-7 stories, ground-floor retail
- **Berlin U-Bahn neighborhoods** (Prenzlauer Berg): 5-6 story perimeter blocks, courtyard gardens
- **Melbourne tram corridors** (Brunswick, Fitzroy): 3-8 story mixed-use along tram routes
- **Portland MAX stations** (Lloyd District, Hollywood): mid-rise TOD in established neighborhoods

## Typology 3: Suburban Center TOD

**Definition**: Commuter rail or end-of-line metro station in a suburban
setting. Opportunity to create a new walkable town center by redeveloping
parking lots, shopping centers, and low-density commercial strips.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | Commuter rail or terminus metro |
| Walk Catchment | 800m |
| Minimum Density | 40 DU/ha (net) |
| FAR Range | 1.5 - 4.0 |
| Height Range | 3 - 8 stories |
| Block Size | 80-120m x 80-120m |
| Street Width (ROW) | Main street 24-30m, locals 15-18m |

### Density Targets

| Zone | FAR | Typical Use |
|---|---|---|
| Station core (0-200m) | 3.0 - 4.0 | Town center: mixed-use, civic, retail, office |
| Inner ring (200-400m) | 2.0 - 3.0 | Mid-rise residential and mixed-use |
| Middle ring (400-600m) | 1.0 - 2.0 | Low-rise apartments, townhouses |
| Outer ring (600-800m) | 0.5 - 1.0 | Townhouses, single-family transition to existing fabric |

### Use Mix

- Residential: 55-65%
- Retail / F&B: 10-15%
- Office: 5-10%
- Civic / Community: 10-15% (town hall, library, school, community center, health clinic)
- Open space: 15-20% of gross area (town green, community park, playfields)

### Street Types

- **Main street**: 24-28m ROW, on-street parking, wide sidewalks with outdoor dining, street trees, mixed-use buildings at 3-5 stories
- **Residential avenue**: 20-24m ROW, tree-lined, bike lanes, front-yard setbacks
- **Local lane**: 12-15m ROW, shared surface, rear access to parking
- **Town green / square**: 0.25-0.5 ha, programmed for markets, events, and daily leisure

### Parking Strategy

- Maximum 0.7 spaces per unit within 400m, 1.0 outside
- Park-and-ride at station (structured, not surface) with 200-500 spaces, reducing over time as development matures
- Shared district parking garages at block interiors, accessed from secondary streets
- On-street parking on main street (parallel, 2-hour limit)
- Phase out surface lots: convert parking lots to development parcels over 10-15 years

### Global Precedents

- **Orenco Station, Portland OR**: LRT TOD in suburban Hillsboro, 1,800 homes, main street retail
- **Fruitvale Village, Oakland CA**: BART station TOD with community focus, health clinic, library
- **Reston Town Center, VA** (evolving toward Metro TOD): suburban center densifying around Silver Line
- **Leidsche Rijn Centrum, Utrecht NL**: new town center around commuter rail, 1,500 homes
- **Cergy-Pontoise, Paris region**: new town around RER stations, town center with civic buildings

## Typology 4: Transit Town Center

**Definition**: New station (often on a new transit line) becomes the catalyst
for creating a complete new neighborhood from scratch. The station and the
town center are planned simultaneously.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | New metro, LRT, or BRT station |
| Walk Catchment | 600-800m |
| Minimum Density | 60 DU/ha (net) |
| FAR Range | 2.0 - 5.0 |
| Height Range | 4 - 15 stories |
| Block Size | 70-100m x 70-100m |
| Street Width (ROW) | Boulevard 28-36m, locals 14-18m |

### Density Targets

| Zone | FAR | Typical Use |
|---|---|---|
| Station core (0-200m) | 4.0 - 5.0 | Civic anchor: library, market hall, co-working, retail |
| Inner ring (200-400m) | 2.5 - 3.5 | Mixed-use residential, office, services |
| Middle ring (400-600m) | 1.5 - 2.5 | Residential with school, park, community facilities |
| Outer ring (600-800m) | 0.8 - 1.5 | Residential, gradually transitioning to lower density |

### Use Mix

- Residential: 55-65%
- Retail / F&B / Market: 10-12%
- Office / Co-working: 5-8%
- Civic / Community: 12-18% (school, library, health center, community center, religious facility)
- Open space: 18-22% (central park, neighborhood greens, playgrounds, community garden)

### Street Types

- **Station boulevard**: 30-36m ROW, transit median (BRT or LRT), protected bike lanes, wide planted sidewalks, mixed-use buildings 6-12 stories
- **Main street**: 22-26m ROW, retail ground floors, outdoor dining, slow traffic
- **Green residential street**: 16-20m ROW, generous tree canopy (50%+ coverage), front gardens or stoops
- **Mews / laneway**: 8-12m ROW, pedestrian-priority, rear access, artists studios, micro-retail
- **Central green / park**: 1.0-2.0 ha, within 400m of all residents, programmed for daily and seasonal use

### Parking Strategy

- Maximum 0.5 spaces per unit within 400m, 0.8 beyond
- District parking at block cores (shared between uses)
- No park-and-ride (this is not a commuter origin station)
- Car-share at 1 per 15 units
- Bike-share distributed throughout at 300m intervals

### Global Precedents

- **Hammarby Sjostad, Stockholm**: tram-served new district, 11,000 homes, closed-loop sustainability
- **Vauban, Freiburg**: tram TOD, car-free district, 5,500 residents, community-led design
- **Aspern Seestadt, Vienna**: U-Bahn extension, 10,500 homes, lake-centered town plan
- **Punggol Waterway, Singapore**: LRT new town, 90,000 residents, waterway park spine
- **King's Cross, London**: new transit-oriented district from brownfield rail lands

## Typology 5: Special District TOD

**Definition**: TOD organized around a major institutional anchor -- hospital,
university campus, airport, sports/entertainment complex, or government center
-- that generates high trip volumes and creates unique demand patterns.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | Metro or LRT, often with dedicated shuttles |
| Walk Catchment | 600-800m (shaped by campus boundaries) |
| Minimum Density | 50 DU/ha (net, excluding institutional campus) |
| FAR Range | 2.0 - 6.0 |
| Height Range | 3 - 15 stories |
| Block Size | Variable (campus blocks may be larger) |
| Street Width (ROW) | Boulevard 28-34m, campus edges 18-24m |

### Density Targets

| Zone | FAR | Typical Use |
|---|---|---|
| Station core (0-200m) | 3.0 - 6.0 | Institutional entry, visitor services, hotel, conference |
| Inner ring (200-400m) | 2.0 - 4.0 | Research/office, student/staff housing, supporting retail |
| Campus edge (400-600m) | 1.5 - 3.0 | Residential (family housing for staff, student housing), neighborhood services |
| Outer ring (600-800m) | 1.0 - 2.0 | Residential, transitioning to existing neighborhood fabric |

### Use Mix (varies by anchor type)

**University Anchor**: student housing 30%, faculty/staff housing 15%, academic/research 20%, retail/F&B 10%, office/incubator 10%, civic/recreation 15%

**Hospital Anchor**: medical office 25%, hospital expansion 15%, patient/visitor housing 10%, staff housing 15%, retail/F&B services 10%, residential 15%, civic 10%

**Airport Anchor**: hotel 25%, office/conference 25%, logistics/light industrial 15%, retail/F&B 15%, residential (noise-compatible zones only) 10%, parking 10%

### Street Types

- **Institutional boulevard**: 28-34m ROW, formal character, drop-off zones, transit stops, wide planted medians
- **Campus edge street**: 18-24m ROW, active frontage on city side, campus wall or gateway on institutional side
- **Supporting commercial street**: 20-24m ROW, services, restaurants, shops serving institutional population
- **Residential street**: 14-18m ROW, quiet, tree-lined, traffic calmed

### Parking Strategy

- Institutional parking consolidated in structured garages at campus periphery (not at station)
- Residential parking maximum 0.5 per unit (institutional populations have low car ownership)
- Visitor parking: structured, shared with evenings/weekends for residential
- Strong TDM: institutional shuttle, subsidized transit passes for employees/students, car-share fleet
- Hospital emergency access maintained separately from general parking

### Global Precedents

- **MIT Kendall Square, Cambridge MA**: Red Line station, research/innovation district, 10M+ sq ft
- **Zurich ETH / University Hospital area**: tram TOD, institutional anchors with mixed-use periphery
- **Singapore one-north (Biopolis)**: MRT station, research campus with integrated housing and retail
- **Copenhagen Rigshospitalet / Panum**: Metro station, hospital campus with research and housing
- **Schiphol Airport, Amsterdam**: rail station, office district, hotel cluster

## Typology 6: Infill TOD

**Definition**: Densification and improvement around an existing station in an
established neighborhood. The station already exists but the surrounding
development has not captured its transit potential. Common in aging inner
suburbs with surface parking lots, low-rise commercial, and single-family
zones near transit.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | Existing metro, LRT, or commuter rail station |
| Walk Catchment | 600-800m (constrained by existing fabric) |
| Minimum Density | 60 DU/ha (net, on redevelopment sites) |
| FAR Range | 2.0 - 5.0 (on redevelopment parcels) |
| Height Range | 4 - 12 stories (on redevelopment parcels), existing fabric retained elsewhere |
| Block Size | Existing (improve with pedestrian passages where possible) |
| Street Width (ROW) | Existing (widen sidewalks, add bike lanes within existing ROW) |

### Development Strategy

This typology is incremental. It does not demolish and replace the entire
neighborhood but surgically identifies underperforming parcels for
redevelopment.

**Priority redevelopment sites (in order)**:
1. Surface parking lots adjacent to station
2. Single-story commercial / strip malls within 400m
3. Gas stations and auto-oriented uses within 400m
4. Underutilized lots (assessed value of improvements < assessed value of land)
5. Aging commercial buildings (50+ years, poor condition, low occupancy)
6. Single-family parcels on corner lots (assemblage for mid-rise)

### Density Targets

| Site Type | Target FAR | Typical Redevelopment |
|---|---|---|
| Surface parking lot at station | 4.0 - 5.0 | Mixed-use with structured parking below |
| Strip mall within 200m | 3.0 - 4.0 | Mid-rise mixed-use with retail ground floor |
| Commercial parcel 200-400m | 2.0 - 3.0 | Mid-rise residential with neighborhood services |
| Assembled parcels 400-600m | 1.5 - 2.5 | Courtyard apartments or townhouses |
| Corner lot 600-800m | 1.0 - 1.5 | Missing middle: duplex, triplex, fourplex, small apartment |

### Use Mix

- Residential: 60-70% (strong emphasis on housing production near transit)
- Retail / F&B: 8-12% (neighborhood-serving)
- Office / Co-working: 5-8%
- Community: 5-10% (upgrade existing or provide new community spaces)

### Street Types (improvements within existing ROW)

- **Existing arterial**: add protected bike lanes by removing a travel lane (road diet), widen sidewalks, add street trees
- **Existing collector**: add bike lanes, improve crossings, plant street trees, activate ground floors
- **Existing local street**: traffic calm to 30 km/h, add sidewalks where missing, add street trees
- **New pedestrian connections**: mid-block passages through redevelopment sites linking station to neighborhoods

### Parking Strategy

- Eliminate parking minimums within 800m of station
- Redevelopment sites: maximum 0.5 spaces per unit within 400m
- Convert existing surface parking to development: build structured parking for existing demand, develop freed land
- Implement on-street parking management (meters, time limits) to manage spillover
- Shared parking agreements between new and existing uses

### Global Precedents

- **Arlington County, VA (Rosslyn-Ballston Corridor)**: 40 years of infill TOD along Metro Orange Line
- **Denver RTD Station Areas**: light rail station area plans transforming suburban stations
- **Sydney Metro Northwest**: existing suburban fabric densifying around new metro stations
- **Vancouver SkyTrain Corridors** (Joyce-Collingwood, Metrotown): incremental densification around stations
- **Minneapolis Greenline LRT**: corridor infill with affordable housing focus

## Typology 7: Greenfield TOD

**Definition**: New transit station and new community built simultaneously on
previously undeveloped or agricultural land. Requires careful planning to
avoid creating another car-dependent suburb that happens to have a train
station.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | New metro extension, LRT, or commuter rail |
| Walk Catchment | 800m (plan for full catchment from day one) |
| Minimum Density | 50 DU/ha (net), increasing to 80+ as community matures |
| FAR Range | 1.5 - 5.0 |
| Height Range | 3 - 12 stories |
| Block Size | 70-100m x 70-100m (design new grid) |
| Street Width (ROW) | Boulevard 28-34m, collectors 20-24m, locals 14-18m |

### Critical Success Factors

Greenfield TOD is the highest-risk typology because there is no existing
urban fabric, market, or community to anchor the development. Success requires:

1. **Transit commitment first**: line must be funded and under construction before development launches
2. **Public realm first**: build parks, streets, and station plaza before or concurrent with first buildings
3. **Civic anchors early**: school, library, or community center in Phase 1 to establish community
4. **Critical mass**: Phase 1 must deliver enough density (minimum 2,000 residents) to support basic retail
5. **Temporary activation**: use temporary uses (markets, pop-up retail, food trucks, events) until permanent retail is viable

### Density Targets

| Zone | Phase 1 FAR | Mature FAR | Typical Use |
|---|---|---|---|
| Station core (0-200m) | 3.0 - 4.0 | 4.0 - 5.0 | Mixed-use center: retail, civic, office, hotel |
| Inner ring (200-400m) | 2.0 - 3.0 | 3.0 - 4.0 | Mid-rise residential, live-work, co-working |
| Middle ring (400-600m) | 1.0 - 2.0 | 1.5 - 2.5 | Low-to-mid-rise residential, school, park |
| Outer ring (600-800m) | 0.5 - 1.0 | 0.8 - 1.5 | Townhouses, row houses, community gardens |

### Use Mix

- Residential: 60-70%
- Retail / F&B: 8-12% (phased: minimal in early phases, growing as population supports it)
- Office / Employment: 5-10%
- Civic / Community: 10-15% (front-loaded in Phase 1: school, library, health clinic)
- Open space: 20-25% of gross area (central park, sports fields, natural areas, community gardens)

### Street Types

- **Station boulevard**: 30-34m ROW, transit in center, bike lanes, double row of trees, mixed-use 6-10 stories
- **Neighborhood main street**: 22-26m ROW, retail ground floors, outdoor dining, on-street parking
- **Green residential avenue**: 18-22m ROW, generous tree canopy, front gardens, bike lanes
- **Home zone / woonerf**: 10-14m ROW, shared surface, play streets, rear-lane access
- **Greenway**: 15-25m wide, multi-use path, bioswale, trees, connecting neighborhoods to station and parks

### Parking Strategy

- Phase 1: maximum 0.8 spaces per unit (residents need cars initially due to limited services)
- Phase 2: reduce to 0.5 per unit as retail and services mature
- Phase 3: reduce to 0.3 per unit as transit frequency increases and neighborhood is self-sufficient
- All parking structured from day one (no surface lots, even temporarily)
- Car-share from Phase 1: 1 per 20 units, growing demand will increase this

### Global Precedents

- **Hammarby Sjostad, Stockholm**: 11,000 homes on brownfield, tram-served, eco-district
- **Aspern Seestadt, Vienna**: 10,500 homes, U-Bahn extension, phased development around lake
- **Punggol New Town, Singapore**: 96,000 homes, LRT served, waterway park spine
- **Almere Oosterwold, Netherlands**: self-build greenfield with planned rail connection
- **Sejong City, South Korea**: new administrative capital with BRT-oriented development

## Typology 8: Corridor TOD

**Definition**: A linear transit corridor (BRT, LRT, or tram) with multiple
station nodes, each developing its own TOD character. The corridor is planned
as a connected system of walkable centers linked by high-frequency transit.

### Design Parameters

| Parameter | Value |
|---|---|
| Transit Mode | BRT, LRT, or tram (linear corridor) |
| Walk Catchment per Station | 400-600m |
| Station Spacing | 600-1,200m |
| Corridor Length | 5-30 km |
| Minimum Density at Node | 50 DU/ha |
| FAR Range | 1.5 - 5.0 (varies by node hierarchy) |
| Height Range | 3 - 12 stories (varies by node hierarchy) |

### Node Hierarchy

Not all stations along a corridor are equal. Establish a 3-tier hierarchy:

**Primary Node (1-3 per corridor)**: interchange station, major destination, highest density
- FAR: 3.0 - 5.0
- Height: 8 - 12 stories
- Character: mini-CBD, major retail, office, civic anchors
- Station plaza: 0.25 ha minimum

**Secondary Node (3-5 per corridor)**: neighborhood center, moderate density
- FAR: 2.0 - 3.5
- Height: 4 - 8 stories
- Character: neighborhood main street, daily services, mid-rise residential
- Neighborhood square: 0.1 ha minimum

**Tertiary Node (remaining stations)**: residential stop, lower density
- FAR: 1.0 - 2.0
- Height: 3 - 5 stories
- Character: residential with corner retail, community facilities
- Pocket park: 0.05 ha minimum

### Corridor-Scale Strategies

- **Green spine**: continuous landscaped corridor along the transit ROW, 6-10m wide, connecting all nodes with a pedestrian and cycling greenway
- **Use differentiation**: distribute uses across nodes (e.g., office concentrated at primary nodes, residential at secondary and tertiary, community facilities distributed)
- **Density rhythm**: highest density at primary nodes, lower between nodes (avoid continuous high-rise walls along the corridor)
- **Identity**: each node should have a distinct character, name, and public space (avoid homogeneous corridor development)

### Street Types

- **Transit corridor**: 30-36m ROW, center-running BRT/LRT, protected bike lanes on both sides, wide sidewalks, station platforms every 600-1200m
- **Node main street (perpendicular to corridor)**: 22-26m ROW, retail ground floors, leading from station into neighborhood
- **Residential feeder street**: 16-20m ROW, connecting residential areas to station
- **Back lane / service street**: 8-12m ROW, rear servicing, reducing conflicts on main streets

### Parking Strategy

- Differentiate by node tier: primary 0.3/unit, secondary 0.5/unit, tertiary 0.8/unit
- No park-and-ride at any corridor station (this is an urban transit corridor, not a commuter line)
- District parking at block interiors, shared between uses
- Corridor-wide car-share and bike-share system with vehicles/bikes rebalancing along the route

### Global Precedents

- **Curitiba BRT Corridors**: pioneering BRT-TOD with structural axes, trinary road system
- **Bogota TransMilenio Corridors**: BRT with station-area densification plans
- **Strasbourg Tram Network**: tram lines restructured development patterns across the city
- **Portland MAX Corridors**: Blue/Red/Orange/Yellow lines with TOD nodes at stations
- **Melbourne tram corridors** (Sydney Road, Smith Street): linear mixed-use intensification
- **Nantes BRT / Tram corridors**: French model of corridor-based urban renewal
- **Istanbul Metrobus Corridor**: 52 km BRT corridor carrying 800,000 daily passengers

## Typology Selection Matrix

Use this matrix to quickly identify the appropriate typology:

```
Is there an existing transit station?
  |
  +--> YES --> Is there significant existing urban fabric?
  |              |
  |              +--> YES --> Is a major institution the anchor?
  |              |              |
  |              |              +--> YES --> Typology 5: Special District TOD
  |              |              +--> NO  --> Is the area CBD/downtown?
  |              |                             |
  |              |                             +--> YES --> Typology 1: Urban Core TOD
  |              |                             +--> NO  --> Is redevelopment the strategy?
  |              |                                           |
  |              |                                           +--> YES --> Typology 6: Infill TOD
  |              |                                           +--> NO  --> Typology 2: Urban Neighborhood TOD
  |              |
  |              +--> NO  --> Typology 3: Suburban Center TOD
  |
  +--> NO  --> Is it a new corridor with multiple stations?
                 |
                 +--> YES --> Typology 8: Corridor TOD
                 +--> NO  --> Is there existing urban fabric nearby?
                                |
                                +--> YES --> Typology 4: Transit Town Center
                                +--> NO  --> Typology 7: Greenfield TOD
```
