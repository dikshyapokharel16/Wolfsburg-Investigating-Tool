# Cycling Infrastructure Design Reference

## Protected Intersection Design

### Dutch-Style Protected Intersection

The gold standard for cycling safety at junctions. Separates all conflict
points and provides clear sightlines.

**Key elements:**
1. **Corner refuge island** - Physical island at each corner separating cycle track from roadway
2. **Setback crossing** - Cycle crossing set back 5-6m from vehicle stop line
3. **Forward stop bar** - Cyclists stop ahead of turning vehicles for visibility
4. **Tight corner radii** - Vehicle turning radius 4-6m (forces slow turns)
5. **Yield markings** - Vehicles turning right yield to through-cyclists

**Dimensions:**
- Corner island depth: 2.0-3.0m
- Setback distance: 5.0-6.0m from motor vehicle stop line
- Cycle crossing width: 3.0-4.0m (bidirectional)
- Pedestrian crossing width: 2.5-4.0m
- Turning radius for motor vehicles: 4.0-6.0m (inner), 8.0-12.0m (outer)

### Signal-Protected Intersection

**Phases:**
1. North-south motor vehicles + parallel cyclists (through only)
2. East-west motor vehicles + parallel cyclists (through only)
3. Dedicated all-turn phase for cyclists (optional, for high-conflict turns)

**Cycle-specific signal elements:**
- Low-level cycle signal heads (2.5m height) for dedicated phases
- Leading pedestrian/cycle interval (LPI): 4-7 seconds head start
- Conflict-free left turn pocket for cyclists
- Advanced stop line (ASL): 4-5m ahead of motor vehicle stop line

### Roundabout Design for Cyclists

**Priority roundabout (Dutch model):**
- Cycle track circles roundabout at 5m offset from vehicle circulatory
- Cyclists have priority at each arm (vehicles entering/exiting yield)
- Raised crossing at each arm
- One-way cycle track: 2.0m wide minimum
- Best for: roundabouts with < 25,000 ADT

**Non-priority roundabout:**
- Shared circulatory with motor vehicles (single lane only)
- Cycle lane: 1.8m painted within circulatory
- Only acceptable at mini-roundabouts with < 8,000 ADT and < 30 km/h

**Grade-separated roundabout:**
- Cyclist tunnel or bridge around roundabout
- Best for: roundabouts with > 25,000 ADT or multi-lane
- Tunnel: minimum 3.5m wide, 2.5m clear height, adequate lighting (100 lux)

## Cycle Track Design Details

### One-Way Protected Cycle Track

| Element | Minimum | Recommended | High Volume |
|---------|---------|-------------|-------------|
| Track width | 1.8m | 2.0-2.5m | 2.5-3.0m |
| Buffer to road | 0.3m (physical) | 0.6-1.0m | 1.0-1.5m |
| Buffer to sidewalk | 0.0m (level) | 0.3m (tactile strip) | 0.5m (planting) |
| Vertical separation | 50mm curb | 75-150mm curb | Full grade separation |
| Surface | Smooth asphalt | Colored asphalt (red/green) | Colored asphalt |
| Cross-slope | 1-2% | 2% (outward) | 2% (outward) |
| Longitudinal grade | < 5% | < 3% sustained | < 3% sustained |

### Two-Way Protected Cycle Track

| Element | Minimum | Recommended | High Volume |
|---------|---------|-------------|-------------|
| Track width | 2.5m | 3.0-3.5m | 4.0-5.0m |
| Center line | Dashed white | Solid white (curves) | Solid white |
| Buffer to road | 0.5m (physical) | 1.0-1.5m | 1.5-2.0m |
| Buffer to sidewalk | 0.3m | 0.5m | 1.0m (planting) |

**Two-way tracks should only be used:**
- On one side of a road (not both sides)
- Where destinations are predominantly on one side
- Along waterfronts, parks, or corridors without frequent cross-streets

### Cycle Superhighway

| Element | Standard |
|---------|----------|
| Width (bidirectional) | 4.0-5.0m |
| Width (unidirectional) | 2.5-3.5m |
| Surface quality | Premium smooth asphalt, colored |
| Gradient | < 3% (max 5% for < 200m) |
| Curve radius | > 20m (min 10m at reduced speed) |
| Intersection priority | Cyclists have priority or grade-separated |
| Lighting | 20+ lux continuous |
| Speed limit | 30-40 km/h design speed |
| Wayfinding | Distance/time signs every 500m |
| Maintenance | Priority snow clearance, daily sweeping |
| Width at passing points | 5.0m minimum |

## Cycle Parking Design

### Short-Term Parking (Visitor)

**Sheffield stand (inverted-U):**
- Dimensions: 750mm wide, 750mm high, mounted 30mm above ground
- Spacing: 1,000mm center-to-center (pairs), 2,500mm aisle
- Each stand parks 2 bikes
- Material: galvanized steel, 48mm OD tube
- Mounting: in-ground or bolt-down

**Placement rules:**
- Within 15m of building entrance
- Visible from street (natural surveillance)
- Under canopy or shelter preferred
- On hard surface (not grass)
- Not blocking pedestrian clear zone (1.8m minimum passthrough)

### Long-Term Parking (Resident / Employee)

**Secure cage / room:**
- Minimum 1.8m2 per bicycle (horizontal)
- Two-tier racks: 1.2m2 per bicycle
- Aisle width: 1.5m minimum (2.0m preferred)
- Door: secure access (key card, code)
- Lighting: 100 lux minimum
- CCTV recommended
- Power outlets for e-bike charging (1 per 5 spaces)

**Cycle locker (individual):**
- Dimensions: 2.0m x 0.8m x 1.2m
- Secure individual lock
- Weather-protected
- Best for: stations, employment areas

### E-Bike Charging

- Standard outlet: 240V / 3A (EU) or 120V / 15A (US)
- Charging time: 3-6 hours for full charge
- Provision: 1 outlet per 5 long-term cycle parking spaces (minimum)
- Location: adjacent to parking, protected from weather
- Metering: coin-operated or smart card for public facilities

## Network Connectivity Standards

### Cycling Network Density

| Context | Target km of cycle route per km2 |
|---------|----------------------------------|
| City center | 3.0-5.0 |
| Inner urban | 2.0-3.0 |
| Outer urban | 1.0-2.0 |
| Suburban | 0.5-1.5 |

### Gap Analysis Protocol

1. Map all existing cycling infrastructure by type (protected, buffered, shared, off-road)
2. Identify all origins (residential areas) and destinations (employment, retail, schools, transit)
3. Calculate cycling isochrones (5, 10, 15, 20 min) from key origins
4. Identify missing links: where the protected network has gaps requiring mixing with high-speed traffic
5. Prioritize gap closure by: demand potential x safety severity x cost efficiency

### Detour Factor Assessment

```
Cycling Detour Factor = Cycling Route Distance / Straight-Line Distance

Target: < 1.2 for all O-D pairs within 5km
Acceptable: 1.2-1.4
Poor: > 1.4 (cyclists will use unofficial routes)
```

## Cycling Level of Service (CLoS)

Based on TfL Cycling Level of Service methodology:

### Factor 1: Safety

| Score | Criteria |
|-------|----------|
| 5 | Fully protected from motor traffic (physical separation) at all points |
| 4 | Protected at high-speed/high-volume roads; shared only on quiet streets (< 20 km/h, < 2,000 ADT) |
| 3 | Buffered lanes on moderate roads; shared on low-traffic streets |
| 2 | Painted lanes on busy roads; occasional mixing with traffic |
| 1 | No infrastructure; mixing with high-speed/high-volume traffic |

### Factor 2: Directness

| Score | Criteria |
|-------|----------|
| 5 | Detour factor < 1.1; cyclist priority at all junctions |
| 4 | Detour factor 1.1-1.2; signal priority at most junctions |
| 3 | Detour factor 1.2-1.3; some junction delays |
| 2 | Detour factor 1.3-1.5; frequent stops at signals |
| 1 | Detour factor > 1.5; forced to use indirect routes |

### Factor 3: Comfort

| Score | Criteria |
|-------|----------|
| 5 | Smooth surface, gentle grades, adequate width, weather protection |
| 4 | Good surface, manageable grades, adequate width |
| 3 | Acceptable surface, some steep sections, narrow at points |
| 2 | Poor surface, steep grades, frequently too narrow |
| 1 | Very poor surface, dangerous grades, width inadequate |

### Factor 4: Attractiveness

| Score | Criteria |
|-------|----------|
| 5 | Green corridor, excellent lighting, active frontage, social safety |
| 4 | Good environment, adequate lighting, mostly pleasant |
| 3 | Mixed environment, some unpleasant sections |
| 2 | Industrial/hostile environment, poor lighting |
| 1 | Unpleasant, unsafe-feeling, poorly maintained |

### Factor 5: Coherence

| Score | Criteria |
|-------|----------|
| 5 | Continuous network, consistent design, clear wayfinding |
| 4 | Minor gaps, mostly consistent, adequate wayfinding |
| 3 | Some gaps, inconsistent infrastructure types |
| 2 | Significant gaps, confusing transitions |
| 1 | Fragmented network, no wayfinding, abrupt endings |

**Overall CLoS** = Average of 5 factors
- 4.0+: Excellent (will attract new cyclists)
- 3.0-3.9: Good (comfortable for regular cyclists)
- 2.0-2.9: Adequate (only for confident cyclists)
- < 2.0: Poor (deters most potential cyclists)

## Bike-Share System Design

### Station Density

| Context | Stations per km2 | Bikes per Station |
|---------|-------------------|-------------------|
| City center | 10-15 | 20-40 |
| Inner urban | 5-10 | 15-25 |
| Neighborhood | 3-5 | 10-20 |
| Campus | 5-8 | 10-15 |

### Station Sizing

```
Docks per Station = Peak Demand x Turnover Factor / Operating Hours x 1.5 (buffer)

Rule of thumb:
  Small station: 12-16 docks (residential area)
  Medium station: 20-30 docks (commercial area, transit stop)
  Large station: 35-50 docks (major transit hub, tourist area)
```

### Station Placement

- Within 30m of transit stops (intermodal transfer)
- At major intersections and destinations
- Spacing: 250-400m between stations in core areas
- Visible from pedestrian routes
- On sidewalk or in parking lane (2.0m x 6-15m footprint)
- Solar-powered where possible (avoid utility connections)

### Fleet Management

- Bikes per 1,000 population: 10-30 (depending on mode share target)
- Rebalancing: 1 vehicle per 50-100 stations
- Maintenance: 1 mechanic per 100-150 bikes
- Average trips per bike per day: 3-8 (successful system)
- Average trip duration: 15-25 minutes
- Average trip distance: 2-4 km
