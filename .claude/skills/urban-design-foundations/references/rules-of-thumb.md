# Rules of Thumb - Quick Lookup by Design Situation

This document provides ready-to-use rules of thumb organized by specific design situations. Each section gives the key metrics, dimensions, ratios, and targets for a particular design task. Use this as a rapid reference when making design decisions or checking proposals.

---

## WHEN DESIGNING A NEIGHBORHOOD CENTER

A neighborhood center is the mixed-use heart of a residential neighborhood, typically located at the intersection of primary pedestrian routes and/or at a transit stop. It should feel distinctly more urban, active, and public than the surrounding residential fabric.

| Parameter | Value |
|---|---|
| FAR | 2.0-4.0 |
| Building height | 4-8 stories (podium + tower if above 6) |
| Use mix | 40-50% residential, 25-35% commercial/office, 10-15% civic/institutional, 10-20% open space |
| Parking (residential) | 0.5-1.0 spaces per unit |
| Parking (commercial) | 2.0-3.0 spaces per 1,000 sqft, with transit proximity reduction of 25-50% |
| Street type | Main Street or Boulevard, H:W ratio 1:1 to 1:2 |
| Block size | 60-80m x 80-100m, perimeter under 400m |
| Intersection density | 100-140 per km2 |
| Active ground floors | Minimum 60% of ground-floor frontage on primary streets dedicated to retail, commercial, or civic uses |
| Ground-floor height | 4.0-5.0m clear height for commercial flexibility |
| Setback | 0m on primary frontages; 0-3m on secondary streets |
| Tree canopy target | 30-40% coverage of public realm |
| Public space | At least one significant plaza (minimum 0.25 ha) plus linear green connections |
| Seating | 1 seat per 10 m2 of plaza, mix of formal (benches) and informal (ledges, steps) |
| Transit | Bus stop within 200m of center; rail within 800m if available |
| Bike parking | Minimum 1 space per 200 m2 of commercial floor area |
| Service/loading | Rear access; no service entries on primary pedestrian frontages |

## WHEN DESIGNING A RESIDENTIAL NEIGHBORHOOD

A residential neighborhood is a primarily housing-focused area that should nonetheless include sufficient local services, green space, and community facilities to support daily life within walking distance.

| Parameter | Value |
|---|---|
| Density | 40-80 DU/ha (net residential); lower end for family housing, higher for apartments |
| Building height | 3-5 stories (maintaining human scale per Alexander Pattern 21 and Gehl) |
| FAR | 1.0-2.5 |
| Use mix | 70-80% residential, 10-15% local retail and community services, 15-20% open space (including streets) |
| Primary school | Within 500m walking distance; 1 per 5,000-10,000 population; 1-2 ha site |
| Secondary school | Within 1,000m; 1 per 15,000-25,000 population; 3-5 ha site |
| Neighborhood park | Within 500m of all residents; minimum 0.5 ha |
| Pocket parks | Within 200m; minimum 0.04 ha each |
| Grocery store | Within 400m of all residents |
| Transit stop | Bus stop within 400m of all residents |
| Street width | 15-20m ROW for local streets; 20-27m for collectors |
| Block size | 60-100m x 80-150m; perimeter under 500m |
| Street speed | 30 km/h design speed on all local streets |
| Parking | 0.8-1.5 per unit; integrated into blocks (rear courtyards, under-building, side lots); no surface parking facing streets |
| Tree-lined streets | 100% of streets with shade trees, spacing 8-10m |
| Sidewalks | Minimum 1.8m clear (ADA), recommended 2.4m, with 1.2m furnishing/tree zone |
| Lighting | 5-8 lux on sidewalks, 10-15 lux at intersections |
| Community facilities | Community center or meeting space within 800m |
| Childcare | Within 400m for family-oriented neighborhoods |
| Building orientation | Long axis E-W in temperate/cold climates for solar gain; adjusted for tropical/arid per climate zone |
| Privacy | 18-22m window-to-window distance between facing habitable rooms |
| Courtyard | Minimum 21m across for daylight access |

## WHEN DESIGNING A MAIN STREET / HIGH STREET

A main street is the primary commercial and social spine of a neighborhood, designed for pedestrian priority with slow-moving vehicles, active ground floors, and a vibrant street life.

| Parameter | Value |
|---|---|
| Right-of-way width | 20-27m total |
| Travel lanes | 2 (one each direction), 3.0-3.3m each |
| On-street parking | Parallel both sides, 2.4m wide |
| Sidewalks | 3.6-4.8m each side (wider where outdoor dining is expected) |
| Furnishing zone | 1.2-1.8m (trees, lights, benches, bike racks) |
| Tree spacing | 8-10m, medium-canopy species for continuous shade |
| Design speed | 30 km/h (enforced by design: narrow lanes, tight corners, on-street parking, raised crossings) |
| Building setback | 0m (build-to line, continuous frontage) |
| Building height | 2-5 stories (H:W ratio 1:2 to 1:1 for good enclosure) |
| Ground-floor use | 80%+ retail, food service, or civic; no residential or parking on ground floor facing main street |
| Ground-floor transparency | Minimum 60% glazing between 0.6m and 2.4m above grade |
| Ground-floor ceiling height | 4.0-5.0m clear for commercial flexibility and signage |
| Door frequency | Minimum 1 entrance per 10m of frontage (Gehl A-grade active frontage) |
| Signage | Projecting blade signs perpendicular to facade, lit at night |
| On-street parking turnover | 1-2 hour time limits; metered where appropriate |
| Intersection spacing | 60-100m (short blocks for pedestrian permeability) |
| Crosswalk frequency | At every intersection; raised crosswalks at minimum every 100m |
| Bike parking | Racks every 30-50m on both sides of street |
| Street furniture | Benches every 60-100m, waste receptacles every 50m, lighting every 15-25m |
| Loading/service | Rear access only; no loading docks on main street frontage |
| Awnings/canopies | Encouraged on all commercial frontages for weather protection |

## WHEN DESIGNING A BOULEVARD

A boulevard is a wide, tree-lined thoroughfare carrying higher vehicle volumes while maintaining a pedestrian-friendly character through generous sidewalks, planted medians, and protected bicycle facilities.

| Parameter | Value |
|---|---|
| Right-of-way width | 30-45m total |
| Central median | 3.6-6.0m, landscaped with trees; can include transit (BRT or LRT) |
| Travel lanes | 2-3 each direction, 3.0-3.3m each |
| Protected bike lanes | 2.0m + 0.6m buffer each side (one-way separated) |
| Sidewalks | 3.6-6.0m each side, with separate frontage, clear, and furnishing zones |
| Tree canopy | Double row on each side (outer row in furnishing zone, inner row at curb or median) plus median trees |
| Design speed | 40-50 km/h (enforced through lane width, signal timing, and street trees) |
| Building height | 4-12 stories (higher at transit nodes, lower at residential transitions) |
| Building setback | 0-3m (build-to zone for continuous street wall; minor setback for entry forecourts) |
| Ground-floor use | Mixed: commercial at nodes, residential between nodes (with appropriate buffer) |
| Median crossing | At-grade pedestrian crossings every 100-150m with refuge islands |
| Transit integration | Center-running BRT or LRT in median where applicable |
| Service roads | Optional 6-7m service lanes parallel to main carriageway for local access, parking, and loading |
| Street lighting | Pedestrian-scale lighting (4-5m height) on sidewalks; roadway lighting (8-12m) for travel lanes |
| Stormwater | Bioswales in median and furnishing zones where feasible |
| Noise mitigation | Tree rows and building setbacks buffer residential from traffic noise; target <55 dB(A) at facade |

## WHEN SIZING A PUBLIC PLAZA

A public plaza is a hard-surfaced, programmed open space designed for gathering, events, rest, and social interaction. Getting the size right is critical: too small and it cannot function for civic use; too large and it feels empty and uncomfortable.

| Parameter | Value |
|---|---|
| Minimum dimension | 15m across (Alexander Pattern 61) |
| Comfortable civic dimension | 25-60m across |
| Maximum before emptiness | 100m across (beyond this, spaces feel deserted without extraordinary programming or crowds) |
| Preferred proportions | Square (1:1) or golden ratio (1:1.6); avoid long, narrow proportions (>1:3) |
| Enclosure | Buildings on minimum 3 sides, minimum 2 stories high; building height = 0.5-1.0x plaza width (Sitte) |
| Seating quantity | 1 linear meter of seating per 3 m2 of plaza area; or 1 seat per 10 m2 |
| Seating types | Mix of fixed benches (with backs, armrests), moveable chairs (where management allows), ledges, steps, planter edges, low walls |
| Seating placement | At edges (edge effect per Gehl), facing activity, in both sun and shade, in pairs and groups (not only rows) |
| Shade | 40-60% of paved area shaded at solar noon in summer (through trees, canopies, structures, adjacent building shadows) |
| Trees | 1 tree per 25-50 m2 of plaza area (more for shade-dominant plazas, fewer for event/paving-dominant plazas) |
| Water feature | Consider for plazas larger than 500 m2; fountains, runnels, splash pads serve as social magnets and white noise |
| Programming | Minimum 10 things to do (PPS Power of 10): sit, eat, play, watch, read, meet, shop, perform, exercise, learn, celebrate |
| Adjacent active uses | Cafes, restaurants, shops, cultural venues on minimum 60% of plaza perimeter |
| Ground-floor transparency | Minimum 70% glazing on facades facing the plaza |
| Wind protection | Avoid wind tunnels from adjacent tall buildings; wind speed at seated height should not exceed 4 m/s for comfort |
| Microclimate | South-facing orientation preferred in temperate/cold climates for sun access; sheltered from prevailing cold winds |
| Surface materials | Durable, non-slip, well-drained, varied textures to define zones; permeable paving where feasible |
| Accessibility | Level or gently graded (<2%); tactile guidance; accessible seating; no steps without ramp alternatives |
| Lighting | 15-30 lux for safety and evening activation; accent lighting on trees, art, and architectural features |
| Drainage | Crown or cross-slope of 1-2% toward drain channels; avoid ponding areas on primary circulation routes |
| Monument/art placement | Off-center per Sitte; visible from primary approach routes; scaled to plaza size; does not block primary use areas |
| Street entries | Via corners rather than wide openings in building wall (Sitte turbine principle); narrow entries intensify the contrast between street and plaza |

## WHEN ORIENTING BUILDINGS

Building orientation significantly affects energy performance, daylight quality, outdoor comfort, and relationship to public space. Correct orientation varies by climate zone and latitude.

**Northern Hemisphere, Mid-Latitudes (30-50 degrees N):**

| Parameter | Guidance |
|---|---|
| Long building axis | East-West orientation for maximum southern facade exposure |
| Primary street grid | Primary streets running E-W, secondary streets N-S |
| South facade | Maximum glazing (40-60%); receives winter sun for passive heating and daylighting |
| North facade | Reduced glazing (20-30%); minimal direct sun; best for diffuse light (studios, offices) |
| East facade | Morning sun; good for bedrooms and breakfast areas |
| West facade | Minimize glazing (20-35%); receives harsh afternoon sun causing overheating; deep overhangs or external shading required |
| Parks and open space | Place on south side of tall buildings to avoid casting them in shadow |
| Courtyards | Open to south; walls on north, east, west for shelter |
| Wind breaks | Orient perpendicular to prevailing cold winter winds |
| Solar access between buildings | Spacing = 1.5-2.5x building height at ground level for winter sun access |

**Southern Hemisphere:** Mirror all orientations (north-facing instead of south-facing, etc.)

**Tropical Climates (within 23.5 degrees of equator):**

| Parameter | Guidance |
|---|---|
| Long building axis | Orient perpendicular to prevailing breeze for maximum cross-ventilation |
| Minimize east/west exposure | Sun at low angle in morning/evening causes deep penetration and overheating |
| Cross-ventilation | Buildings narrow enough (max 14m depth) for effective natural ventilation |
| Deep overhangs | 1.5-2.5m on all facades for rain and sun protection |
| Elevation | Raise buildings on pilotis for air circulation underneath |
| Vegetation | Dense tree canopy on east and west sides for shading |

**Hot-Arid Climates:**

| Parameter | Guidance |
|---|---|
| Street orientation | Primary streets E-W (narrow, shaded by buildings on both sides) |
| Building mass | Thick walls, high thermal mass for temperature damping |
| Courtyards | Inward-facing buildings around shaded courtyards |
| Minimize east/west facade area | Greatest heat gain from low-angle sun |
| Surface color | Light colors (albedo >0.5) for roofs and walls |
| Wind | Orient wind catchers toward prevailing cooling breezes |
| Shade structures | Continuous arcades, pergolas, and shade canopies over pedestrian routes |

## WHEN CALCULATING POPULATION AND SERVICES

These formulas and ratios allow quick estimation of population yield from a development and the community facilities required to serve that population.

### Population Yield Calculation

```
Total population = Net residential area (ha) x Dwelling density (DU/ha) x Average household size
```

| Average household size by type | Persons |
|---|---|
| Studio / 1-bedroom apartment | 1.3-1.5 |
| 2-bedroom apartment | 2.0-2.5 |
| 3-bedroom apartment / townhouse | 2.8-3.2 |
| 4+ bedroom family house | 3.5-4.5 |
| Blended average (mixed development) | 2.2-2.8 |

### Community Facility Requirements

| Facility | Population Threshold | Site Area | Service Radius | Notes |
|---|---|---|---|---|
| Primary school (elementary) | 1 per 5,000-10,000 | 1.0-2.0 ha | 500m (walk) | Child yield varies: apartments 0.2-0.5 children/unit; houses 0.5-1.5 |
| Secondary school (middle/high) | 1 per 15,000-25,000 | 3.0-5.0 ha | 1,000m (walk/cycle) | May serve multiple neighborhoods |
| Health clinic / doctor's surgery | 1 per 10,000-20,000 | 0.1-0.3 ha | 800m | Larger clinics serve larger populations |
| Community center / hall | 1 per 10,000-15,000 | 0.2-0.5 ha | 800m | Multi-use: meetings, classes, events, social services |
| Public library | 1 per 20,000-30,000 | 0.2-0.5 ha | 1,200m | Can be combined with community center |
| Place of worship | Varies by culture | 0.1-1.0 ha | Varies | Typically 1 per 5,000-10,000 in many cultures |
| Fire station | 1 per 20,000-40,000 | 0.3-0.5 ha | 3 km response | Response time target: <5 minutes |
| Police station | 1 per 40,000-60,000 | 0.2-0.5 ha | District-wide | Neighborhood policing offices may be smaller and more frequent |
| Post office | 1 per 10,000-20,000 | 0.05-0.1 ha | 800m | Increasingly co-located with retail |
| Childcare / nursery | 1 per 3,000-5,000 | 0.1-0.3 ha | 400m | Higher frequency in family-oriented neighborhoods |
| Sports / recreation center | 1 per 15,000-25,000 | 1.0-3.0 ha | 1,500m | Indoor and outdoor facilities |
| Elderly care / day center | 1 per 15,000-25,000 | 0.2-0.5 ha | 800m | Co-locate with health and community services |
| Pharmacy | 1 per 5,000-10,000 | Retail unit | 400m | Often market-provided in mixed-use areas |
| Supermarket / grocery | 1 per 5,000-10,000 | 0.2-0.5 ha | 400m | Anchor for neighborhood center; 1,000-3,000 m2 floor area |
| Convenience store | 1 per 2,000-3,000 | Retail unit | 200m | Often market-provided |

### Employment Yield Estimation

| Use Type | Employment Density | Notes |
|---|---|---|
| Office (general) | 1 job per 12-20 m2 NIA | Higher density = modern open plan; lower = traditional |
| Office (creative/tech) | 1 job per 15-25 m2 NIA | More generous per-person allocation |
| Retail (general) | 1 job per 15-25 m2 GFA | Includes storage and back-of-house |
| Restaurant / cafe | 1 job per 10-15 m2 GFA | Higher labor intensity |
| Light industrial | 1 job per 25-50 m2 GFA | Varies widely by industry |
| Warehouse / logistics | 1 job per 50-100 m2 GFA | Increasingly automated |
| Hotel | 1 job per 1-2 rooms | Depends on service level |

## WHEN PLANNING INFRASTRUCTURE

Quick estimates for infrastructure demand and land allocation in new developments.

### Land Budget for a Typical Neighborhood (40-60 ha)

| Land Use Category | Percentage of Total Area | Notes |
|---|---|---|
| Residential (net) | 35-45% | All residential parcels including private gardens/courtyards |
| Commercial / mixed-use | 5-10% | Neighborhood center, high street, local retail |
| Civic / institutional | 3-8% | Schools, community center, health, worship |
| Streets and infrastructure | 25-35% | All public rights-of-way including sidewalks (UN-Habitat: 30-35%) |
| Parks and open space | 15-25% | All public green space including plazas, playgrounds, sports |
| Parking (if surface/structured) | 0-5% | Lower in transit-rich areas; zero for well-designed TOD |

### Utility Demand Summary

| Utility | Per Person Per Day | Per Dwelling Per Day | Notes |
|---|---|---|---|
| Water supply | 150-250 liters | 350-600 liters | Climate and lifestyle dependent |
| Wastewater generation | 120-225 liters | 280-540 liters | 80-90% of water supply |
| Solid waste | 0.5-1.5 kg | 1.2-3.5 kg | Varies by income level and waste policy |
| Electricity | 2-8 kWh | 5-20 kWh | Excluding heating/cooling in some climates |
| Heating (cold climate) | 15-30 kWh | 35-75 kWh | Seasonal; depends on building efficiency |
| Cooling (hot climate) | 5-15 kWh | 12-35 kWh | Seasonal; depends on building efficiency |

### Street Infrastructure Allocation

| Component | Percentage of Street ROW | Notes |
|---|---|---|
| Pedestrian zone (both sidewalks) | 30-40% of ROW | Higher percentage on local/neighborhood streets |
| Cycle facilities | 10-15% of ROW | Where protected lanes are provided |
| Travel lanes | 25-35% of ROW | Minimum necessary; avoid over-allocation |
| Parking | 10-15% of ROW | On-street parallel or angled; can be zero on some streets |
| Furnishing / planting | 10-20% of ROW | Trees, lighting, drainage, utilities |
| Median | 0-15% of ROW | On boulevards and divided streets only |

## WHEN DESIGNING FOR TRANSIT INTEGRATION

Rules of thumb for integrating land use and transit at various scales.

### Walk Catchment by Transit Mode

| Transit Mode | Walk Catchment Radius | Comfortable Walk Time | Population Needed to Support |
|---|---|---|---|
| Local bus (15-30 min headway) | 400m | 5 minutes | 5,000-10,000 |
| Frequent bus (<10 min headway) | 500m | 6-7 minutes | 10,000-20,000 |
| BRT (Bus Rapid Transit) | 600-800m | 8-10 minutes | 15,000-30,000 |
| Light rail / tram | 800m | 10 minutes | 25,000-50,000 |
| Heavy rail / metro | 800-1,200m | 10-15 minutes | 50,000-100,000+ |
| Commuter rail | 1,200-1,600m | 15-20 minutes | 20,000-50,000 at station area |

### Density Gradient from Transit Station

| Distance from Station | Recommended Density (DU/ha) | Recommended FAR | Typical Height |
|---|---|---|---|
| 0-200m (station area) | 150-400+ | 4.0-10.0+ | 6-20+ stories |
| 200-400m (inner ring) | 80-200 | 2.5-5.0 | 4-10 stories |
| 400-800m (outer ring) | 40-100 | 1.5-3.0 | 3-6 stories |
| 800-1,200m (edge) | 25-60 | 0.8-2.0 | 2-4 stories |

### Parking Reduction Near Transit

| Distance from Frequent Transit | Parking Ratio Reduction |
|---|---|
| 0-200m | 50-75% reduction from standard |
| 200-400m | 30-50% reduction |
| 400-800m | 15-30% reduction |
| >800m | Standard ratios apply |

## WHEN ASSESSING WALKABILITY

Quick diagnostic checklist for evaluating whether an area is walkable.

### Physical Infrastructure Checklist

| Element | Minimum Standard | Good Standard | Excellent Standard |
|---|---|---|---|
| Sidewalk presence | Both sides of all streets | Both sides, continuous, no gaps | Both sides, wide (>3m), well-maintained |
| Sidewalk surface | Paved, level | Smooth, even, non-slip | High-quality materials, universally accessible |
| Crossings | At all signalized intersections | At all intersections + mid-block on long blocks | Raised crossings, short wait times (<30s), direct desire lines |
| Curb ramps | At all crossings (ADA) | Flush with detectable warning surface | Continuous accessible route throughout |
| Lighting | All intersections and transit stops | Continuous on both sidewalks (5+ lux) | High-quality, pedestrian-scale (15+ lux on commercial streets) |
| Street trees | Some presence | 50%+ of block faces | Continuous canopy both sides (80%+ coverage) |
| Benches / rest points | At transit stops | Every 200m | Every 100m, with backs and armrests |
| Wayfinding | Major destinations signed | Comprehensive pedestrian signage | Digital wayfinding, tactile maps, multilingual |

### Network Quality Checklist

| Metric | Poor | Adequate | Excellent |
|---|---|---|---|
| Intersection density | <60 per km2 | 80-100 per km2 | 120+ per km2 |
| Average block perimeter | >600m | 400-500m | <400m |
| Cul-de-sac percentage | >20% of street segments | 5-10% | <5% |
| Route directness ratio | >1.5 (actual/straight line) | 1.2-1.3 | <1.2 |
| Dead-end streets | >15% of segments | 5-10% | <5% |
| Mid-block passages | None | On blocks >120m | On all blocks >80m |

## WHEN DESIGNING FOR CLIMATE RESILIENCE

Quick rules of thumb for integrating climate resilience into urban design.

### Heat Island Mitigation

| Strategy | Temperature Reduction | Notes |
|---|---|---|
| Tree canopy (per 10% increase) | 0.5-1.0 degrees C | Evapotranspiration + shade; most effective strategy |
| Green roofs | 0.3-0.8 degrees C at roof level | Extensive green roofs on >50% of roof area |
| Cool roofs (high albedo) | 0.3-0.5 degrees C | Albedo >0.65 for significant effect |
| Cool pavements | 0.2-0.5 degrees C | Light-colored or permeable; albedo >0.3 |
| Water features | 1.0-3.0 degrees C locally | Evaporative cooling; most effective in dry climates |
| Reducing impervious surface | 0.2-0.5 degrees C per 10% reduction | Replace asphalt with vegetation or permeable paving |

### Flood Resilience

| Strategy | Sizing Guidance | Notes |
|---|---|---|
| Finished floor level above flood | Minimum 0.5m above 100-year flood level | 1.0m freeboard recommended for climate change |
| On-site detention | Limit post-development peak flow to pre-development rate | Size for 10-100 year storm depending on jurisdiction |
| Permeable paving area | 30-50% of parking and pedestrian hardscape | Not suitable for heavy vehicle traffic areas |
| Rain garden sizing | 5-7% of contributing impervious area | Size for first 25mm of rainfall |
| Green corridor width | Minimum 10m for ecological function | 30-50m for significant flood conveyance and habitat |
| Building setback from waterway | Minimum 10m; recommended 30m | Varies by flood risk and ecological sensitivity |

### Wind Comfort in Tall Building Areas

| Wind Speed at Pedestrian Level | Comfort Level | Acceptable For |
|---|---|---|
| <4 m/s | Comfortable for sitting | Outdoor dining, reading, relaxing |
| 4-6 m/s | Comfortable for standing | Window shopping, waiting, socializing |
| 6-8 m/s | Comfortable for walking | Transit stops, walking routes |
| 8-10 m/s | Uncomfortable for most activities | Tolerable for short-duration passage only |
| >10 m/s | Dangerous | Unsafe; requires design mitigation |

**Mitigation strategies:** podium bases on towers (3-6 stories before setback to tower), canopies and arcades at ground level, porous screens and perforated walls, dense vegetation, building orientation perpendicular to prevailing wind, staggered building arrangements to break wind patterns.
