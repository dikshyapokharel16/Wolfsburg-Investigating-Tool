# Mitigation Strategies -- Complete Reference

This reference provides a comprehensive catalog of urban heat island
mitigation strategies, green infrastructure specifications, urban tree
selection guidance, cool materials databases, and passive building envelope
strategies at the urban scale.

---

## 1. Green Infrastructure Catalog

### 1.1 Bioretention Systems

**Bioretention Cell (Rain Garden)**

| Parameter | Specification |
|---|---|
| Surface area | 5-7% of contributing impervious area |
| Ponding depth | 150-300mm |
| Soil media depth | 600-900mm |
| Soil media composition | 50-60% sand, 20-30% compost, 10-20% topsoil; hydraulic conductivity 25-50 mm/hr |
| Gravel storage layer | 200-300mm washed aggregate (40mm nominal) |
| Underdrain | 150mm perforated PVC at base of gravel layer; required where native soil infiltration <15 mm/hr |
| Overflow | riser pipe or spillway set at maximum ponding depth |
| Drain time | 24-48 hours maximum from full ponding |
| Setback from buildings | 3m minimum from foundations |
| Setback from trees | 3m minimum from trunk (to protect roots) |
| Pollutant removal | TSS: 85-95%, phosphorus: 50-80%, nitrogen: 40-60%, metals: 80-95%, bacteria: 60-90% |

**Engineered Bioswale**

| Parameter | Specification |
|---|---|
| Width (bottom) | 0.6m minimum, 1.2m typical, 2.4m maximum |
| Depth | 150-300mm ponding + 100-200mm freeboard |
| Side slopes | 3:1 maximum (4:1 preferred for mowing access) |
| Longitudinal slope | 1-2.5% (use check dams at 15-25m intervals on steeper sites) |
| Flow velocity | <0.3 m/s during design storm |
| Soil media | same as bioretention cell if infiltration function is primary; or 150mm topsoil over native for conveyance-focused design |
| Check dams | stone, timber, or concrete; 150-200mm high, full width, keyed 200mm into side slopes |
| Inlet | curb cuts (150mm wide x 75mm deep) at 3-5m intervals; or continuous curb opening |
| Vegetation | dense planting (80%+ coverage at maturity): native grasses, sedges, iris, canna, ferns |

**Stormwater Planter**

| Parameter | Specification |
|---|---|
| Configuration | Contained planter box with impermeable walls and bottom; drains via underdrain to storm sewer |
| Use case | Where infiltration is not feasible (high water table, contaminated soil, adjacent to foundations) |
| Dimensions | 0.6-1.5m wide, 0.6-0.9m deep (visible above grade), variable length |
| Soil media | same as bioretention; 600mm minimum depth |
| Ponding | 150-200mm above soil surface |
| Drain time | 12-24 hours |
| Location | along building edges, in tree wells, integrated into sidewalk furnishing zone |
| Structural | walls designed for lateral soil and water pressure; waterproof membrane on building side |

### 1.3 Detention and Retention Systems

**Dry Detention Basin**

| Parameter | Specification |
|---|---|
| Volume | Sized for design storm (10-100 year depending on jurisdiction) minus any upstream green infrastructure credit |
| Depth | 0.6-2.4m maximum ponding depth |
| Side slopes | 4:1 or flatter for safety and mowing |
| Outlet | restricted orifice sized to release stored volume over 24-72 hours |
| Emergency spillway | designed for 100-year event; vegetated or armored |
| Bottom | flat or gently sloped (1-2%) to low-flow channel; permeable surface preferred |
| Forebay | sediment forebay at inlet (15-25% of total volume) for pre-treatment |
| Landscape | mowed turf or native meadow planting; trees on berms (not on embankment) |

**Wet Retention Pond**

| Parameter | Specification |
|---|---|
| Permanent pool volume | equal to the water quality volume (first 25mm x contributing area) |
| Wet pool depth | 1.0-2.5m permanent depth |
| Aquatic bench | 0-300mm depth, 2-5m wide, around pond perimeter (safety, habitat, aesthetics) |
| Littoral zone planting | emergent aquatic plants on aquatic bench: cattails, bulrush, pickerelweed |
| Buffer | minimum 8m vegetated buffer from property lines |
| Aeration | fountain or mechanical aerator recommended to prevent algae and odor |
| Amenity | can serve as park feature with pathways, seating, fishing |

**Underground Detention**

| Parameter | Specification |
|---|---|
| Types | Concrete vaults, HDPE chambers (e.g., StormTech), large-diameter pipe |
| Volume | same sizing as dry detention basin |
| Access | manholes at every 30m for inspection and cleaning |
| Maintenance | annual inspection, jet-cleaning every 3-5 years |
| Use case | where surface area is not available (urban infill, under parking lots, plazas) |
| Structural | designed for traffic loads if under pavement; minimum 0.6m cover |

### 1.4 Green Roofs

**Extensive Green Roof**

| Parameter | Specification |
|---|---|
| Substrate depth | 40-150mm |
| Saturated weight | 60-150 kg/m2 |
| Vegetation | Sedum species, mosses, drought-tolerant grasses, low-growing herbs |
| Irrigation | not required in temperate and tropical; drip irrigation for establishment in hot-arid |
| Slope | 0-25 degrees (anti-shear layer required above 10 degrees) |
| Drainage layer | 25-40mm egg-crate or geocomposite |
| Root barrier | polyethylene or TPO root barrier; copper-impregnated fabric optional |
| Stormwater retention | 50-70% of annual rainfall |
| Thermal performance | reduces cooling load by 10-30%; reduces heating load by 5-10% |
| Maintenance | 2-4 inspections per year, weeding 1-2 times per year, fertilizing once per year |

**Intensive Green Roof**

| Parameter | Specification |
|---|---|
| Substrate depth | 150-1000mm+ |
| Saturated weight | 200-1500 kg/m2 |
| Vegetation | full landscape palette: perennials, shrubs, trees (with adequate depth and root volume) |
| Irrigation | required in all climates except tropical wet |
| Structural | requires structural engineer coordination; significant additional dead and live load |
| Stormwater retention | 70-90% of annual rainfall |
| Access | designed as occupiable roof garden; railings, pathways, lighting, drainage |
| Maintenance | similar to ground-level landscape: weekly during growing season |

**Blue-Green Roof**

| Parameter | Specification |
|---|---|
| Configuration | extensive or intensive green roof with additional sub-surface water storage (detention) layer |
| Storage layer | 40-80mm depth below green roof substrate; controlled-flow outlet (orifice or smart valve) |
| Stormwater retention | 85-95% of annual rainfall |
| Smart controls | IoT-connected valve that pre-drains storage before predicted rain events (improves capacity by 30-50%) |
| Cost premium | 15-25% above standard green roof |

## 2. Urban Tree Species Selection Guide

### Selection Criteria Decision Framework

```
Step 1: Climate Zone
  |
  +--> Hot-Arid: drought-tolerant species, deep roots, minimal water
  +--> Tropical: evergreen or briefly deciduous, high transpiration, rapid growth
  +--> Temperate: deciduous (for seasonal light control), moderate water
  +--> Cold: cold-hardy (check USDA zone), salt-tolerant near roads
  |
Step 2: Site Constraints
  |
  +--> Soil volume available? (<10 m3: small tree; 10-20 m3: medium; >20 m3: large)
  +--> Overhead clearance? (power lines: max 8m height; clearance needed for trucks: 4.5m+)
  +--> Underground utilities? (shallow utilities: use deep-rooting species, root barriers)
  +--> Salt exposure? (road de-icing: select salt-tolerant species)
  +--> Pollution exposure? (heavy traffic: select pollution-tolerant species)
  |
Step 3: Design Intent
  |
  +--> Maximum shade: large canopy species (10-15m spread), dense leaf
  +--> Winter sun access: deciduous with fine branching (90%+ light when bare)
  +--> Windbreak: dense evergreen, planted in rows 2-3 deep
  +--> Visual screen: columnar or fastigiate form
  +--> Seasonal interest: flowering, fall color, bark texture
  +--> Wildlife habitat: fruit or nut bearing, native species
  +--> Stormwater management: high transpiration rate, tolerant of wet feet
```

### Soil Volume Standards

Adequate soil volume is the single most important factor determining urban
tree survival and growth. Trees planted in insufficient soil volume will
decline, fail to reach mature canopy size, and require early replacement.

| Tree Size Category | Canopy Spread | Minimum Soil Volume | Preferred Soil Volume |
|---|---|---|---|
| Small (ornamental) | <6m | 8 m3 | 12 m3 |
| Medium | 6-10m | 15 m3 | 20 m3 |
| Large | 10-15m | 20 m3 | 30 m3 |
| Extra-large (specimen) | >15m | 30 m3 | 50 m3 |

**Soil volume provision methods**:
- **Open tree pit**: 1.5m x 1.5m minimum opening (poor practice; rarely sufficient)
- **Continuous tree trench**: 1.2-1.8m wide trench connecting multiple trees, shared soil volume
- **Structural soil cells (Silva Cells)**: modular frames supporting pavement above while providing 100% usable soil volume below; industry best practice for street trees
- **Structural soil (CU-Soil / Stockholm method)**: gap-graded stone matrix with soil in voids; supports pavement loads, 20-25% usable soil volume
- **Suspended pavement**: pavement slab on piers or beams over uncompacted soil; 100% usable volume

## 4. Building Envelope Strategies at Urban Scale

### Passive Cooling Strategies

**Natural Ventilation**
- Effective when outdoor temperature <32 C and humidity <70%
- Cross-ventilation: building depth <15m with operable windows on opposite facades
- Stack ventilation: atria, stairwells, or dedicated shafts (minimum 1 m2 opening per 50 m2 floor area at top and bottom)
- Night purge: open building to cool night air (effective when diurnal range >8 C); pre-cool thermal mass for next day
- Wind-driven: orient building openings to capture prevailing wind; inlet area should be smaller than outlet area (2:3 ratio) for acceleration

**Evaporative Cooling (Hot-Arid Only)**
- Direct evaporative: wetted pads, misting systems, fountains; reduces temperature by 8-15 C when RH <40%
- Indirect evaporative: evaporative cooling of supply air through heat exchanger (no moisture added to indoor air); effective to 25-28 C supply air
- Urban-scale: water channels in public spaces, misting corridors, fountain plazas; effective in 50m radius
- Water budget: misting system uses 5-10 L/m2/hour; must be sourced from recycled or non-potable water

**Earth Cooling**
- Ground temperature at 3m depth: approximately equal to annual average air temperature (stable to within 2 C)
- Earth tubes: outdoor air drawn through buried pipes (100-200mm diameter, 20-50m length, 2-3m depth) to pre-cool by 5-10 C before entering building
- Earth-sheltered construction: bermed or partially underground buildings; roof and walls in contact with earth provide stable temperature and reduce cooling/heating loads by 30-50%

**Shading Systems (Urban Scale)**
- Tensile shade structures: PTFE or PVC membrane; 10-30 year lifespan; can span 10-50m; commonly used over streets, plazas, and parking in Gulf cities
- Retractable canopies: mechanized fabric canopies that deploy seasonally or by time of day
- Photovoltaic shade structures: PV panels as shade canopies over parking, plazas, walkways; dual function: energy generation + shade
- Vegetation-based: deciduous trees (seasonal shade), climbing plants on pergolas (seasonal or evergreen), green walls

### Passive Heating Strategies

**Direct Solar Gain**
- South-facing windows (northern hemisphere): the primary passive heating strategy
- Glazing area: 7-12% of floor area as south-facing glazing for passive solar homes; higher ratios need thermal mass to prevent overheating
- Thermal mass: 6 m2 of 100mm-thick concrete or masonry per m2 of south glazing to absorb and store solar heat
- Night insulation: insulated shutters or blinds (R-4+) closed at night reduce heat loss through glazing by 60-80%

**Trombe Wall**
- Masonry or concrete wall (200-400mm thick) behind south-facing glazing with 50-100mm air gap
- Performance: provides 15-30% of heating demand in temperate climates; negligible benefit below 35 degrees latitude
- Modern variants: include operable vents for daytime convective heating and nighttime dampers to prevent reverse flow
- Application at urban scale: south-facing party walls, retaining walls, sound walls along transit can function as Trombe walls for adjacent buildings

**Sunspace / Conservatory**
- Glazed room on south facade: acts as solar collector and thermal buffer
- Temperature range: 5-20 C warmer than exterior in winter (unheated)
- Sizing: 10-20% of adjacent floor area
- Operation: open to building during sunny winter days (heat gain); close at night (thermal buffer, reducing heat loss by 30-50%)

**Wind Protection (Urban Scale)**
- Evergreen windbreaks: 3 rows of staggered planting, 50% porosity; reduce wind speed by 50-70% for a distance of 10-15x tree height downwind
- Earth berms: 2-3m high, planted, along exposed edges; effective wind shelter for parks and residential areas
- Building arrangement: L-shaped or U-shaped building groups open to the south, closed to prevailing cold wind; creates sheltered courtyard
- Underground connections: heated pedestrian tunnels between buildings at major nodes (Montreal RESO model, Minneapolis Skyway)

## 5. Climate Change Adaptation

Urban designs created today will experience significantly different climate
conditions over their 50-100 year lifespan. The following adjustments account
for projected climate change impacts.

### Temperature Projections (2050, RCP 4.5)

| Current Zone | Projected Change | Design Implication |
|---|---|---|
| Hot-Arid | +1.5 to 2.5 C | Design for more extreme heat events; increase shade and cooling infrastructure; drought will intensify |
| Tropical | +1.0 to 2.0 C | Higher humidity extremes; increase ventilation capacity; more intense rainfall events |
| Temperate | +1.5 to 2.5 C | Shift toward warmer conditions; increase summer cooling capacity; more extreme precipitation events |
| Cold | +2.0 to 4.0 C (highest warming) | Reduced heating demand but increased cooling demand; permafrost thaw; more freeze-thaw cycles in shoulder seasons |

### Rainfall Projections (2050, RCP 4.5)

| Region | Change | Design Implication |
|---|---|---|
| Arid regions | -10 to -30% annual (more severe drought) | Increase water harvesting capacity by 30%; design for zero external water input |
| Tropical regions | +5 to +20% annual; +20-40% intensity of extreme events | Upsize all stormwater infrastructure by 30%; design for flash flooding; elevate ground floors |
| Temperate regions | Variable; +10-30% winter, -10-20% summer; more intense individual events | Design stormwater for 1.3x current 100-year event; drought-resilient species selection |
| Cold regions | +10 to +30% annual (more as rain, less as snow) | Rain-on-snow flooding risk; earlier spring melt; design drainage for rain+melt compound events |

### Adaptive Design Strategies

- **Oversizing green infrastructure**: design bioswales and detention for 1.3x current design storm (accounts for intensity increase without cost of full redesign)
- **Species diversification**: plant a diverse palette of tree species (no single species >10% of total) to hedge against climate-related pest and disease risk
- **Flexible shading**: retractable or seasonal shade systems that can be expanded as temperatures increase
- **Dual-mode buildings**: designed for both heating and cooling, with the capacity to shift emphasis over time
- **Flood-resilient ground floors**: commercial or parking uses at grade in flood-prone areas; residential above first floor; wet-floodproofed utilities
- **Cool surface triggers**: specify contract provisions to re-coat roofs and pavement with cool coatings when temperatures exceed threshold (adaptive maintenance)

## 6. Performance Monitoring

### Key Performance Indicators for Climate-Responsive Design

| KPI | Measurement Method | Target | Frequency |
|---|---|---|---|
| Urban Heat Island intensity | Weather station comparison (urban vs. rural) | <2 C UHI at night | Continuous |
| Canopy coverage | Aerial LiDAR or satellite imagery | 30-40% (varies by zone) | Every 2 years |
| Stormwater runoff volume | Flow meter at site outlet | <30% of pre-development peak | Per storm event |
| Outdoor thermal comfort (UTCI) | Microclimate station (temp, RH, wind, radiation) | UTCI 9-26 C for 70%+ of daylight hours | Continuous |
| Building energy use intensity | Utility meter, normalized by area and climate | <80 kWh/m2/year total | Monthly |
| Permeable surface ratio | Site survey / aerial imagery | >40% of non-building area | Every 5 years |
| Tree survival rate | Annual tree inventory | >90% at 5 years | Annual |
| Water consumption (landscape) | Irrigation meter | <50% of reference ET | Monthly (growing season) |

### Monitoring Equipment

- **Weather station**: temperature, humidity, wind speed and direction, solar radiation, rainfall; at 2m and 10m heights; minimum 2 stations per TOD area (one exposed, one sheltered for comparison)
- **Soil moisture sensors**: capacitance probes at 150mm and 300mm depth in bioretention facilities and tree pits; wireless data logger
- **Flow meters**: at all stormwater outlets and key inlet points; ultrasonic or magnetic type
- **Tree monitoring**: dendrometer bands on sample trees (10% of population) for growth measurement; visual health assessment annually

This monitoring data enables evidence-based adaptive management, allowing
urban managers to adjust strategies -- increasing shade, modifying irrigation,
re-grading bioswales -- based on actual performance rather than assumptions.
